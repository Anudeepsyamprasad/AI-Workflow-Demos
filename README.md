## Prompt Engineering Framework
All workflows follow this structured approach:

1. **Role Definition** – Establish AI’s task role  
2. **Task Instruction** – What to accomplish  
3. **Constraints** – Boundaries & safety requirements  
4. **Output Format** – Expected structure  
5. **Review Layer** – Human confirmation before action  

This pattern improves consistency and reduces risk.

---

## Safe AI Usage Principles
When using AI tools in regulated business environments:

1. Always include human review  
2. Never input confidential or sensitive data  
3. Define clear constraints in prompts  
4. Avoid automated final decisions  
5. Maintain documentation for audit

These principles reduce risk and preserve compliance.
